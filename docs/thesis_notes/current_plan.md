# 当前缩减版复现计划

## 目标
在现有主机性能一般的条件下，完成一个能跑通、能评估、能写进本科毕业论文的缩减版复现项目。

## 当前优先级
1. LiDAR 分支：PointPillars
2. RGB 分支：Faster R-CNN
3. 融合部分：Bbox Matching、Semantic Fusion
4. Detection Recovery：可简化或作为扩展内容

## 当前环境
- Docker 容器：mmcv-container
- 工作目录：/workspace/thesis_hlcf
- 代码基础环境：mmcv + mmdetection3d
- 论文参考：A Multimodal Hybrid Late-Cascade Fusion Network for Enhanced 3D Object Detection
